# Consistent player mapping

An application which tracks and id's players in a video feed making sure the id's are consistent . 
---

## Requirements

* Python **3.9+**
* `pip`
* Virtual environment support

---

## Setup & Installation

### 1. Create and activate a virtual environment

```bash
python -m venv venv
source venv/bin/activate 
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

---

## Running the Application
(Before running please make sure that you also have the model `best.pt` in the same directory as `main.py` . )
```bash
python main.py
```
